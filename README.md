This is a relatively simple project creating some insightful plots based on this LinkedIn dataset: https://www.kaggle.com/datasets/arshkon/linkedin-job-postings
We also used stock data from Yahoo Finance.

Our task was that we had a new AI-based Talent Intelligence startup and we wanted to find a successful, market-leading company in a strong industry with 
long-term demand for hiring skilled labor to act as our anchor client.
